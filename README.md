# 个人IP · 工作经验沉淀

> 记录技术成长，分享实践经验，打造个人技术品牌。

## 关于

这个仓库用于沉淀我的工作经验和技术思考，内容同步输出到微信公众号等图文平台。涵盖技术复盘、项目实战、工具效率、职业成长等方面。

## 目录结构

```
readme/
├── articles/                  # 文章目录
│   ├── project-practice/     # 项目实战 — 真实项目经验复盘
│   ├── tech-review/          # 技术复盘 — 技术选型、踩坑记录
│   ├── tools/                # 效率工具 — 工具推荐、工作流优化
│   └── career/               # 职业思考 — 成长感悟、行业观察
├── templates/                 # 文章模板
│   └── article-template.md   # 默认文章模板
└── README.md
```

## 写作流程

1. 从 `templates/article-template.md` 复制模板
2. 按分类放入 `articles/` 对应目录
3. 撰写完成后修改 `status` 为 `published`
4. 同步发布到微信公众号

## 内容原则

每篇文章力求：

- **真实**：来自实际工作场景
- **深度**：不止于表面，分享背后的思考
- **实用**：提供可落地的经验和方法
